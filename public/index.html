About

Turning complexity
into clarity

I'm a final-year software engineering student at Singapore Institute of Technology, graduating this year and actively looking for graduate roles. I came to software through aerospace engineering — studying high-stakes systems where precision and failure costs are both extreme trained me to reason carefully about problems before reaching for a keyboard.

What drives me is automation. Taking a process that consumes time and mental energy, breaking it down to its essential steps, and rebuilding it in software so a machine handles it reliably — that is where I do my best work. I've applied this to financial reporting, embedded systems test workflows, and image validation pipelines.

I write primarily in Python, with experience in Java and JavaScript, and I care about producing code that is readable, testable, and easy to hand off. If you are building something where software should be doing more of the heavy lifting, I would like to talk.
